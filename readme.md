# Quiz des Capitales du Monde

Un projet Python pour apprendre et tester les capitales par continent.

---

## Description

Ce quiz permet aux utilisateurs de choisir un continent (Afrique, Europe, Amérique, Asie, Océanie) et de répondre à des questions sur les capitales des pays de ce continent.  

Le script gère automatiquement :  
- Les réponses avec majuscules ou minuscules  
- Les accents et espaces dans les réponses  

---

## Fonctionnalités

- Choisir le continent à tester  
- Quiz aléatoire sur les pays du continent choisi  
- Score final à la fin du quiz  
- Prise en charge des variations de saisie (accents, majuscules, espaces)
